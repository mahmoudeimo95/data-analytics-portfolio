# Sales Analytics Project
# Coffee Shop Sales Analytics

## Business Problem

The coffee shop wants to understand sales performance, customer purchasing behavior, product popularity, and payment trends in order to increase revenue and improve operational decisions.

## Tools Used

- Excel
- SQL
- Python (Pandas, Matplotlib)

## Objectives

- Analyze revenue trends
- Identify top-selling products
- Evaluate payment methods
- Discover peak sales periods
- Generate business recommendations
